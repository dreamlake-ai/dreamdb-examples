# DreamDB Examples

Usage examples for [DreamDB](https://dreamdb.dreamlake.ai) with real-world datasets.

## Examples

| Example | Description |
|---------|-------------|
| [`llff-dreamdb/`](llff-dreamdb/) | LLFF scene with Plücker coordinate vector index (DreamDB) |
| [`llff-lancedb/`](llff-lancedb/) | Same LLFF use case using LanceDB for comparison |
| [`llff-rerun/`](llff-rerun/) | Convert LLFF scenes to Rerun (.rrd) + LanceDB for visualization |

## Getting Started

Each example is self-contained. See the README inside each directory for setup instructions.

## Large Files

This repo uses [Git LFS](https://git-lfs.github.com/) for images and binary assets. Run `git lfs install` before cloning.
